# 📥 دليل تثبيت Claude Code

> **الوقت المتوقّع:** 5-10 دقائق
> **المتطلّبات:** حاسوب (Windows / Mac / Linux) + اشتراك Claude Pro أو Max

---

## ⚡ الطريقة الأسرع: التثبيت بأمر واحد

### على نظام Mac أو Linux

افتح **Terminal** وألصق هذا الأمر:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

### على نظام Windows

افتح **PowerShell** كمسؤول (Run as Administrator) وألصق:

```powershell
irm https://claude.ai/install.ps1 | iex
```

---

## 🔍 طريقة بديلة: التحميل اليدوي

1. افتح: **https://claude.com/claude-code**
2. اضغط **Download for [نظامك]**
3. اتبع تعليمات التثبيت

---

## ✅ التحقّق من نجاح التثبيت

بعد انتهاء التثبيت، أغلق Terminal/PowerShell وافتحه من جديد، ثم نفّذ:

```bash
claude --version
```

يجب أن ترى رقم الإصدار (مثل: `claude 1.x.x`).

---

## 🔐 تسجيل الدخول

نفّذ الأمر:

```bash
claude
```

سيفتح متصفّحك تلقائياً لتسجيل الدخول بحساب Claude نفسه (نفس حساب claude.ai).

---

## 📂 بدء العمل مع المشروع

### الخطوة 1: فُكّ ضغط ملف المشروع

1. فُكّ ضغط `sra-warehouse-app.zip` على سطح المكتب.
2. ستحصل على مجلد `warehouse-app`.

### الخطوة 2: افتح المجلد في Terminal

**على Mac/Linux:**
```bash
cd ~/Desktop/warehouse-app
```

**على Windows (PowerShell):**
```powershell
cd $HOME\Desktop\warehouse-app
```

### الخطوة 3: شغّل Claude Code

```bash
claude
```

ستُفتح جلسة تفاعلية مع Claude **داخل مجلد مشروعك**.

## 🆘 حلول المشاكل الشائعة

### المشكلة: `claude: command not found`
- **الحل:** أغلق Terminal/PowerShell وافتحه من جديد.

### المشكلة: التثبيت يفشل على Windows
- **الحل:** تأكّد من فتح PowerShell **كمسؤول** (Run as Administrator).

### المشكلة: لا أرى نافذة تسجيل الدخول
- **الحل:** انسخ الرابط الذي يظهر في Terminal والصقه في المتصفح يدوياً.

### المشكلة: يقول "اشتراك غير صالح"
- **الحل:** تأكّد من تفعيل اشتراك Claude Pro في حسابك على https://claude.ai
