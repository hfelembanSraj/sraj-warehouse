# نظام إدارة المستودعات — جمعية المسؤولية الاجتماعية

نظام متكامل لإدارة مستودعات العُدّة التشغيلية مبني بـ React و Supabase.

## ✨ الميزات الرئيسية

- 🗺️ **منظور علوي** للمستودع بالأبعاد الواقعية (4م × 4م)
- 📦 **نظام ترميز هرمي** للأدوات (مساحة-رف-صندوق)
- 🔄 **إدارة الإخراج والإرجاع** مع مهلة 10 أيام افتراضية
- 🗑️ **تتبّع المتلفات** مع إمكانية الاسترجاع
- 💝 **سجل الدعم** للأدوات المُسلَّمة لجهات
- 👥 **نظام صلاحيات دقيق** (6 صلاحيات مستقلة لكل مستخدم)
- 🤖 **اقتراحات ذكية** لمواقع التخزين
- 📱 **رموز QR** للستيكرات الفيزيائية
- 📊 **تقارير وإحصائيات** شاملة
- 🔐 **سجل تتبّع كامل** لكل العمليات

## 🚀 البدء السريع

اتبع **[دليل النشر](./DEPLOYMENT_GUIDE.md)** خطوة بخطوة.

## 🏗️ البنية التقنية

- **الواجهة:** React 18 + Vite + Tailwind CSS
- **قاعدة البيانات:** Supabase (PostgreSQL + Auth)
- **الاستضافة:** Vercel
- **رموز QR:** qrcode.js

## 📁 هيكل المشروع

```
sra-warehouse/
├── src/
│   ├── components/    # المكوّنات (الجداول، النماذج، التبويبات)
│   ├── pages/         # الصفحات (الدخول، التسجيل، اللوحة)
│   ├── context/       # AuthContext لإدارة المصادقة
│   ├── lib/           # المساعدات والثوابت وعميل Supabase
│   ├── App.jsx        # المكوّن الرئيسي
│   ├── main.jsx       # نقطة الدخول
│   └── index.css      # التنسيقات الرئيسية
├── supabase/
│   └── setup.sql      # سكربت إعداد قاعدة البيانات
├── public/            # الملفات الثابتة
├── DEPLOYMENT_GUIDE.md  # دليل النشر المفصّل
├── package.json
├── vite.config.js
└── tailwind.config.js
```

## 🛠️ التطوير المحلي (اختياري)

```bash
npm install
cp .env.example .env
# أضف مفاتيح Supabase في .env
npm run dev
```
